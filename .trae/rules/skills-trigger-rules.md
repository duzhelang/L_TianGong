# 技能触发规则

## 可用技能

| 技能名称 | 触发条件 | 用途 |
|----------|----------|------|
| ecocarbon-architecture | 询问架构、功能、API、规范、技术细节 | 定位系统架构文档 |
| ecocarbon-data-assets | 询问数据、模型、训练、数据库内容 | 定位数据资产文档 |
| ecocarbon-git | `/git` 指令、提交代码、推送变更、查看 git 状态 | Git 提交推送工作流 |

## 触发示例

### ecocarbon-architecture
- "系统架构是什么？"
- "API有哪些端点？"
- "后端模块怎么划分？"
- "开发规范是什么？"

### ecocarbon-data-assets
- "数据库有哪些表？"
- "数据怎么流转？"
- "排放因子存在哪里？"
- "训练数据在哪里？"

### ecocarbon-git
- "/git"
- "提交代码"
- "推送变更"
- "git 状态"
- "commit"

## 技能调用方式

当用户询问相关问题时，使用 Skill 工具调用对应技能获取文档路径索引。
